Tags h

Voltar para o início

As tags h representam títulos de diversos tamanhos podendo ir de h1 até h6, sendo h1 o título com maior tamanho de fonte e h6 o menor.

Tag ol

Voltar para o início

A tag ol representa uma lista ordenada. Tendo como marcador de cada item um número, podendo ser em ordem crescente ou decrescente. Tags que representam listas são utilizadas em conjunto com a tag li que veremos logo mais.

  1. Primeiro
  2. Segundo
  3. Terceiro

Tag ul

Voltar para o início

Ao contrário da tag ol, a tag ul representa uma lista não ordenada, onde os marcadores de cada item por padrão são representados por pontos. Essa tag também é utilizado em conjunto com a tag li.

  • Item 1
  • Item 2
  • Item 3

Tag li

Voltar para o início

Como visto anteriormente a tag li representa cada item de dentro da lista, sendo comumente utilizada dentro das tags ol ou ul.

Tag a

Voltar para o início

A tag a é utilizada quando queremos criar um link seja para uma outra página ou quando queremos criar uma ancoragem, que seria uma forma de dar um scroll até onde o ponto de ancoragem foi colocado. Como está sendo utilizado no índice dessa página. Vale lembrar que para que a tag a leve até um link ou ancoragem é preciso utilizar dentro dela o atributo href, colocando nele a referência para onde queremos que o link nós leve. Quando utilizamos o href o texto é formatado para a cor azul com um sublinhado, indicando que aquele teste é um link. Exemplo de um texto formatado como link

Tag hr

Voltar para o início

A tag hr serve para criarmos uma linha em nossa página, indicando uma quebra de parágrafo.

Tag sub

Voltar para o início

A tag sub serve para representarmos um texto subscrito. Subscritos são comumente utilizados para escrever elementos químicos por exemplo. H2O

Tag sup

Voltar para o início

Já a tag sup serve para representarmos um texto sobrescrito, fazendo a estilização contrária da tag sub. Sobrescritos são utilizaos por exemplo para escrever fórmulas matématicas. X2 + 2X3

Tag blockquote

Voltar para o início

A tag blockquote é utilizada para indicar uma citação dentro de um texto, ela identa o texto para indicar que aquele bloco textual representa uma citação.

Tag font

Voltar para o início

A tag font é utilizada para estilizar a fonte de um texto, seja a cor, tamho e formato da fonte. Porém essa tag foi descontinuada em alguns dos navegadores sendo recomendado utilziar o atributo font no css.

Tag abbr

Voltar para o início

A tag abbr é uma tag semântica que ajuda o navegador a reconhecer que aquele pedaço de texto é uma abreviação. Normalmente após o uso da tag abbr damos a descrição completa daquela abreviação entre parênteses. por exemplo: HTML (HyperText Markup Language)
